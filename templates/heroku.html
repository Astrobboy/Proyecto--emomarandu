What is Heroku about?

Heroku es una plataforma en la nube que permite a las empresas crear, entregar, supervisar y escalar apps — Es la forma más rápida de ir desde los dolores de cabeza de estructura URL. Las aplicaciones se corren desde un servidor Heroku usando Heroku DNS Server para apuntar al dominio de la aplicación (típicamente nombreaplicacion.herokuapp.com). Cada aplicación corre sobre un motor a través de una “red de bancos de prueba” que consta de varios servidores. El servidor Git de Heroku maneja los repositorios de las aplicaciones que son subidas por los usuarios.
